### The Name "Shim" in Computing

"Shim" is a generic, well-understood computing term:
- **Definition:** A library that transparently intercepts API calls and adds capability
- **Usage:** Windows Application Compatibility Toolkit, JDBC drivers, polyfills
- **Status:** Public domain concept (like "kernel", "driver", "buffer")

This makes it:
- ‚úÖ Immediately meaningful to developers
- ‚úÖ Describes exactly what the product does
- ‚úÖ Not ownable by any single company
- ‚úÖ Easy to remember and spell

### Trademark Considerations

- Generic computing terms generally cannot be trademarked
- Similar to how "Windows" was allowed but contested
- Product-specific trademark possible: "SHIM for Claude Desktop"
